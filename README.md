# Arduino Course

Welcome to the Arduino Course repository! This repository contains a varied collection of code files and projects designed for learning and implementing Arduino components

## 📚 What's Included
This repository is comprehensive and includes code for a variety of hardware, including:
* **Motors** 
* **Sensors** 
* **Modules** 
* **Full Basic Robots**

## 🗂️ Repository Structure
To make learning and navigation easier, this repository is divided into two main levels based on difficulty[cite: 7]:
* **Juniors:** Contains smaller and simpler codes ideal for beginners.
* **Seniors:** Contains bigger and harder codes for advanced learners and more complex projects.

## 📁 File Organization
Each specific component is neatly organized into its own dedicated folder. Within each folder, you will find:
* An **`.ino` file** containing the actual Arduino source code.
* A dedicated **`README.md` file** that explains how the specific component functions and how the code works.
